Modulo 01 - Funções

Lista de Numeros Aleatórios

Desafio: Criar um novo projeto, que dispare um evento controlado para gerar números de forma aleatória entre 0 e 100. Os valores devem ser retornados na tela do usuário na forma de uma lista pontilhada.

Calculadora de Potência

Desafio: Criar um novo projeto, que dispare um evento controlado para calcular a exponenenciação de um número. Peça que o usuário informe a base e o expoente. O valor deve ser mostrado ao usuário.

Melhor Combustível

Desafio: Criar um novo projeto, que dispare um evento controlado para calcular o mais viavel combustível, tendo em mente o preço e consumo por kilômetro. O resultado deve ser mostrado ao usuário.

Coversor de Temperatura

Desafio: Criar um novo projeto, que dispare um evento para converter graus Celsius em graus Fahrenheit e Kelvin de. A conversão deve ser feita de forma automatica, onde devem ser mostradas ao usuário.

Simulador Estatistico

Desafio: Criar um novo projeto, que dispare um evento controlado para gerar uma simulação de dados, com avlores aleatorios, em 6 cenários distintos, em um loopin de repetição de 1.000.000 de veses.

Modulo 02 - Arrays

Valores Extremos do Conjunto

Desafio: Criar um novo projeto, que dispare um evento contolado, para encontrar o maior e menor valor de um conjunto de números, que devem serem informados pelo usuário junto com o separador. O resultado deve ser exibido para o usuário.

Lista de Tarefas

Desafio: Criar um novo projeto, que dispare um evento contolado, para gerenciar uma lista de tarefas, com as funcionalidades: apagar, listar, atualizar e remover. A lista dever ser exibido ao usuário em uma lista pontilada por ordem de criação.

Estante de Filmes

Desafio: Criar um novo projeto, que dispare um evento automatico para filtrar filmes pelo nome, alem de ler um array contendo informações de flimes e lista-lo em uma tabela. Ambas as duas funcionalidades devem ocorrer de forma automatica.

Lista Salarial

Desafio: Criar um novo projeto, que dispare um evento contolado para adionar o salário de um novo funcionario e que gere um relatório com o maior salário adicionado e o valor total de todos os salarios. Os salarios devem ser exibidos em uma tabela.

Modulo 03 - Objetos e Classes

Agenda de Eventos

Desafio: Criar um novo projeto, que dispare um evento controlado para armazenar uma agenda de eventos, cada evento deve conter nome e data. Forneça ao usuário a opção de escolher a data diretamente no calendário. Os dados devem ser apresnetados ao usuário por meio de uma tabela, em ordem cronologica da data.

Conta Bancária

Desafio: Criar um novo projeto, que dispare um evento controlado para armazenar contas bancarias. Cada conta deve ter as opções de depositar e sacar, auterando o saldo de acordo com a movimentação. A movimentação deve ser feita apenas em contas existentes. O dados devem ser informados em uma tabela.

Transação Bancária

Desafio: Criar um novo projeto, que dispare um evento controlado para armazenar transações bancarias. Deposito, para valores positivos e saque, para valores negativos. O resultado deve ser apresentado em uma tabela, e a cor do valor manipulado deve ser equivalente ao tipo da transação, vermelho ou verde.

Modulo 04 - Modularidade e Armazenamento

Sistema de Estoque

Desafio: Criar um novo projeto que dispare umm evento para armazenar produtos no localStorage. Desenvolva as funções: adicioanr, remover, listar, marcar e desmarcar, em um arquivo modulo e as implemente no arquivo principal. Os dados devem ser apresentados em uma interface. Mais especificações na documentação do modúlo 04.
